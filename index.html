Introduction

HTML (HyperText Markup Language) is the foundation of web development, providing a set of markup tags and elements that structure and format content for display on browsers, enabling the creation of interactive and visually appealing web pages.

Elements and tags

Opening tags in HTML indicate the beginning of an element and consist of a keyword enclosed in angle brackets (e.g., <p> for a paragraph). Conversely, closing tags signal the end of an element and are similar to opening tags, but with a forward slash before the keyword (e.g., </p> for closing a paragraph).

Example:

<p>some text context </p>

Start tag Element content End tag
<h1> First header </h1>
<p> Paragraph </p>
<br> none none

NOTE

Some HTML elements have no content (like the <br> element). These elements are called empty elements. Empty elements do not have an end tag!

HTML Basics

All HTML documents must start with a document type declaration: <!DOCTYPE html>.
The HTML document itself begins with <html> and ends with </html>.
The visible part of the HTML document is between <body> and </body>.

Example:

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
